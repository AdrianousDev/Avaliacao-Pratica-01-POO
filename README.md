# Avaliação 01 – Modelagem UML e Implementação de Classes

## Passo a passo para rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/AdrianousDev/Avaliacao-Pratica-01-POO.git
```

### 2. Rode o arquivo Main.java dentro de src

#### Direto pelo terminal

```bash
java src/Main.java
```

#### Ou, coloque os arquivos em sua IDE e rode o Main.java lá
